# Cline Memory Bank (Fataplus & fataplus-app)

Ce dossier est la base de connaissances centralisée pour les deux applications principales du projet :

- **Fataplus** : application fullstack développée avec Nuxthub (Nuxt.js)
- **fataplus-app** : application Next.js, clone du boilerplate whop.com

## Objectifs

- Centraliser les informations techniques, organisationnelles et historiques sur les deux apps.
- Faciliter la transmission de connaissances, l’historique des décisions, et l’ajout de notes ou liens importants pour l’équipe.
- Servir de référence rapide pour tout nouveau membre ou contributeur.

## Structure proposée

- `fataplus-context.md` : Contexte technique, organisationnel et historique de l’app Fataplus (Nuxthub).
- `fataplus-app-context.md` : Contexte technique, organisationnel et historique de l’app fataplus-app (Next.js/whop.com).
- D’autres fichiers pourront être ajoutés selon les besoins (historique, décisions, liens, etc.).
