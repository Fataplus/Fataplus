nice# Cline Memory Bank

Ce dossier sert de base de connaissances ("memory bank") pour le projet Fataplus et ses applications associées.

## Objectif

- Centraliser les informations contextuelles, techniques et organisationnelles sur les applications Fataplus (Nuxt/Nuxthub) et fataplus-app (Next.js/boilerplate Whop.com).
- Faciliter la transmission de connaissances, l’historique des décisions, et l’ajout de notes ou liens importants pour l’équipe.

## Structure proposée

- `fataplus-context.md` : Contexte technique, organisationnel et historique de l’app Fataplus (fullstack, Nuxthub).
- `fataplus-app-context.md` : Contexte technique, organisationnel et historique de l’app fataplus-app (Next.js, clone Whop.com).
- D’autres fichiers pourront être ajoutés selon les besoins (historique, décisions, liens, etc.).
