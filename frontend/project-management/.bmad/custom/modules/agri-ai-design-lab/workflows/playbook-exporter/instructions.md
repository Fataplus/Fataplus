# Playbook Exporter Instructions

<workflow>
  <step n="1" goal="Collect artifacts">
    <action>Gather latest outputs from core and feature workflows (kickoff canvas, concept report, prototype plan, personas, blueprints).</action>
    <action>Confirm file paths and ensure bilingual summaries if requested.</action>
  </step>
  <step n="2" goal="Assemble structure">
    <action>Organize materials into sections: context, insights, concepts, design, build, CX, next steps.</action>
    <action>Embed links to Figma files, no-code projects, and datasets where available.</action>
  </step>
  <step n="3" goal="Generate playbook">
    <action>Populate template placeholders and format according to Fataplus style guide.</action>
    <action>Include summary page, key metrics, and recommended roadmap.</action>
  </step>
  <step n="4" goal="Prepare handoff packet">
    <action>Create file index, deliverable checklist, and communication plan.</action>
    <action>Save final Markdown/Notion export and attach any additional assets.</action>
  </step>
  <step n="5" goal="Output">
    <action>Store generated playbook in {output_folder}/playbooks.</action>
    <action>Provide completion summary with next actions.</action>
  </step>
</workflow>
